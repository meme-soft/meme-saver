# meme-saver

app for managing memes

## Requirements

- NodeJS
- npm
- MongoDB

## Development

### Environment

setup local development envrionment

### Back-end

install dependecies

```shell
cd server
npm install
```

start server with

```shell
cd server
npm start
```

### Front-end client

install dependecies

```shell
cd client
npm install
```

start react app with

```shell
cd client
npm start
```
